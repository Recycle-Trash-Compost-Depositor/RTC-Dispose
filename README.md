
# Recycling Website

ISAAC created this website from scratch, and his biggest challenge was teaching his teammates about HTML.

This is a code bundle for Recycling Website. The original project is available at https://www.figma.com/design/jiZG0KRHXFdaMNRN2FutcD/Recycling-Website.

The pages included in the project are:
1. The Recycling page was made.
2. The Trash page was made.
3. The Compost page was made.
4. The Benefits page was made.
5. The Home page was made.
6. The About Us page was made.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
